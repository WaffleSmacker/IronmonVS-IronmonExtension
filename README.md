# IronmonVS-IronmonExtension

This enables data to be sent straight to the ironmonvs.com site so streamers can focus on gaming.

## Ironmon VS Tracker Extension
This is the official Ironmon VS tracker extension.
The ironmonvs.lua should be placed directly into the extensions folder.
IronmonVsMonitor.exe should be inside of the IronmonVS folder.

### Installation
1. Open your Ironmon Tracker and navigate to Settings > Extensions.<br>
<img src="https://github.com/user-attachments/assets/1e000ec5-89b3-4990-8672-697a0897aa23" width="300" /><br><br>

2. Click the General tab and select "Install a New Extension".<br>
<img src="https://github.com/user-attachments/assets/6844bcef-3072-4595-816e-9612ac87f15a" width="300" /><br><br>

3. Paste the following link into the window:
   `https://github.com/WaffleSmacker/IronmonVS-IronmonExtension`<br>
<img src="https://github.com/user-attachments/assets/2ba0b49d-ad80-465b-8c95-eb91c86b6374"  width="450"/><br><br>


4. Confirm Installation: Verify that IronmonVs appears in your list. To activate it, click the extension and check the enable box.<br>

## IronmonVsMonitor.exe
The Monitor is a standalone program that sends your run data to the website. Using a separate executable prevents emulator lag and allows for better data quality checks.

### Set Up Instructions

1. Launch the Program: Open IronmonVsMonitor.exe.
2. Connect Twitch: Go to Settings and click Connect Twitch.<br>
<img src="https://github.com/user-attachments/assets/b2ceae25-69de-4b80-9d90-84992f9b92c1" width="300" /><br><br>

3. Twitch Authentication: This will open a Twitch OAuth window. Log in through the secure Twitch portal (the program never sees your password).<br>
<img src="https://github.com/user-attachments/assets/76dc6554-b6ec-430e-80df-df5090b1329b" width="200" /><br>
<br>
4. Verify Login: If successful, your Twitch username should now be visible within the program.<br>
<img src="https://github.com/user-attachments/assets/5cf6ab5c-f697-4cee-8520-17227ad07c15" width="300" /><br><br>

5. Test Connection: Click Back and select Send Test Ping to ensure you can communicate with ironmonvs.com.<br>
<img src="https://github.com/user-attachments/assets/33843956-c3f0-4ac6-a457-8215ab2314bd" width="300" /><br><br>

6. Success: A successful ping will return a confirmation message.<br>
<img src="https://github.com/user-attachments/assets/174c4c3c-e5bb-44cc-9abc-fb18cd3ac33d" width="300" /><br><br>

### Using the Monitor
Once configured, simply start your game and grab a Pokemon from the table. Your data will automatically populate in the IronmonVsMonitor!<br>
