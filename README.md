# IronmonVS-IronmonExtension

This enables data to be sent straight to the [ironmonvs.com](https://ironmonvs.com) site so streamers can focus on gaming.

## Overview
This is the official Ironmon VS tracker extension. To function correctly, ensure your files are organized as follows:
* The `ironmonvs.lua` file should be placed directly into your tracker's **extensions** folder.
* `IronmonVsMonitor.exe` should be located inside the **IronmonVS** folder.

## Installation & Setup

### 1. Extension Installation
1.  Open your **Ironmon Tracker** and navigate to `Settings` > `Extensions`.<br>
    <img src="https://github.com/user-attachments/assets/1e000ec5-89b3-4990-8672-697a0897aa23" width="300" /><br><br>

2.  Click the **General** tab and select **"Install a New Extension"**.<br>
    <img src="https://github.com/user-attachments/assets/6844bcef-3072-4595-816e-9612ac87f15a" width="300" /><br><br>

3.  Paste the following link into the window:  
    `https://github.com/WaffleSmacker/IronmonVS-IronmonExtension`<br>
    <img src="https://github.com/user-attachments/assets/2ba0b49d-ad80-465b-8c95-eb91c86b6374" width="300"/><br><br>

4.  **Confirm Installation:** Verify that `IronmonVs` appears in your list. To activate it, click the extension and check the enable box.<br>
    <img src="https://github.com/user-attachments/assets/b997ad31-8f60-4298-b4e8-26ab904516c7" width="300" /><br>
    <img src="https://github.com/user-attachments/assets/200fe54d-d7af-4233-a6cc-e4246e17d58d" width="300"/><br><br>

### 2. Monitor Setup
5.  **Launch the Monitor:** Open `IronmonVsMonitor.exe`. <br>
    *Note: If you cannot find the file, you can also click **"options"** within the IronmonVS extension in your tracker.*<br>
    <img src="https://github.com/user-attachments/assets/1530e088-6590-4863-8c17-a692ee59d543" width="300" /><br><br>

6.  **Connect Twitch:** Go to `Settings` and click **Connect Twitch**.<br>
    <img src="https://github.com/user-attachments/assets/b2ceae25-69de-4b80-9d90-84992f9b92c1" width="300" /><br><br>

7.  **Twitch Authentication:** A Twitch OAuth window will open. Log in via the secure portal.<br>
    <img src="https://github.com/user-attachments/assets/76dc6554-b6ec-430e-80df-df5090b1329b" width="200" /><br><br>

8.  **Verify Login:** Once successful, your Twitch username should be visible within the program.<br>
    <img src="https://github.com/user-attachments/assets/5cf6ab5c-f697-4cee-8520-17227ad07c15" width="300" /><br><br>

9.  **Test Connection:** Click **Back** and select **Send Test Ping**.<br>
    <img src="https://github.com/user-attachments/assets/33843956-c3f0-4ac6-a457-8215ab2314bd" width="300" /><br><br>

10. **Success:** A successful ping will return a confirmation message.<br>
    <img src="https://github.com/user-attachments/assets/174c4c3c-e5bb-44cc-9abc-fb18cd3ac33d" width="300" /><br><br>
